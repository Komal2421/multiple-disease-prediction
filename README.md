# MedPredict 

MedPredict  is a Flask-based web application that employs machine learning algorithms to predict the risk for different illnesses and offers AI-generated health insights through the Gemini API.# Functionalities

- Foretells Diabetes
- Prediction of Heart Disease
- Prediction of Parkinson’s Disease
- Prediction of Breast Cancer
- Authentication of Users
AI-Driven Clinical Insights
- Automatic Best Model Selection # Technology Stack

 - Python 
 - Flask 
- Scikit-learn 
- Pandas 
- NumPy 
- HTML 
- CSS.
- JavaScript 
- Gemini API# Installation ````bash git clone https://github.com/Komal2421/multiple-disease-prediction.git cd multiple-disease-prediction pip set up -r requirements.txt python app.py ```

Create a .env file:
````text GEMINI_API_KEY=YOUR_API_KEY ```

Run the program and go to:
``` http://127.0.0.1:5000 ``
